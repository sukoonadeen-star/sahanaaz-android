Sahanaaz Enterprises - Flutter App (upgraded)
============================================

Included features:
- Polished Material UI (basic)
- Invoice view screen and PDF export (print/share)
- Local SQLite storage using sqflite
- GST calculation: Bihar = CGST+SGST, others = IGST
- Company details pre-filled:
    Business name: Sahanaaz Enterprises
    Address: Khagra stadium Kishanganj Bihar
    Authorized signatory: Samsher Alam

Run:
1. Install Flutter SDK.
2. Open the project folder in Android Studio or VS Code.
3. Run: flutter pub get
4. Run on device/emulator: flutter run
Note: This is a starter app. Customize styling, assets, and tests before production.
